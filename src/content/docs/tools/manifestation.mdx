manifestation is a tool to make publishing GDWeave mods easier.

<LinkButton href="https://github.com/NotNite/manifestation">GitHub</LinkButton>

## Usage

See [the README](https://github.com/NotNite/manifestation/blob/main/README.md) for more information.

To use manifestation, double click the executable and answer the asked questions. Then, when you want to build your project, drag the `manifestation.toml` onto the executable.
